## Distributions

To start with, we're going to need a source of randomness. For this, I'm going
to use [RandN](https://randn.dev/), a NuGet package providing alternative random
number generation to the built-in System.Random. The main thing I like about
RandN is how it defines an interface `IDistribution<T>`, which represents a
source of random `T`s (for some type `T`, e.g. `int`) corresponding to some
probability distribution.

```cs
public interface IDistribution<T>
{
    T Sample<TRng>(TRng rng);
}
```

(Note: there's also a `TrySample()` method on this interface, which we will be
largely ignoring during this series.)

## Manipulating distributions

functor and monad to get normal distribution

```cs
public static class Normal
{
    public static IDistribution<double> New()
    {
        var uniformDist = Uniform.New(0d, 1d);
        return
            from u1 in uniformDist
            from u2 in uniformDist
            select
                Math.Sqrt(-2.0 * Math.Log(u1))
                * Math.Cos(2.0 * Math.PI * u2);
    }
}
```
